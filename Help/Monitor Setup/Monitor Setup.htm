Monitor Setup

This consists of two forms in sequence relating to the setting up of monitors for first use in PrgLnch.

Verify Device Numbers:
This dialog shows up when there exist two or more monitors attached to the desktop. They must be all switched on and connected to the adapter in order to be accessible through Window Settings, the driver settings, and PrgLnch. Click the Monitor buttons so the monitor names match the monitor on which they are shown. If the monitor names are the same, no adjustment is required.
Missing Stock Resolutions:
Upon first launch, a popup list of resolutions may show. In the case where more than one monitor is detected on the system, it appears following the above device number verification. These resolutions are detected by WMI for the device, but do not match those in the stock list. They may be available later when custom resolutions are set up in the adapter driver user interface.

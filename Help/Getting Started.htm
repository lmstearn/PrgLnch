Getting Started

Preliminaries

As reference tools, the question mark on the titlebar of a PrgLnch window along with the F1 key are provided for context help on the various form items.


Note: The following discussion is based on a Windows 10 install, there may be small differences in other versions of Windows.

Start PrgLnch, and If there is more than one monitor in the system, verify Device Numbers before continuing. Also keep a record of any Missing Stock Resolutions, if so desired.
Once the PrgLnch gui shows, first check, and please report if the controls on the form are clipped (usually a result of the font being other than MS Shell Dlg at 8 size), as not all font combinations have been tested. PrgLnch uses files on your disk, so an idea of the approximate location of programs or shortcuts to be used in PrgLnch is obtained by a quick search in a File Explorer window. For best results in any file search, rebuild its index.

Navigating Through PrgLnch

PrgLnch opens with the PrgLnch Batch form, where batched programs are to be listed. Not much here yet, so, click Prg Config on the bottom RHS, and proceed to the drop-down list on the Upper LHS where an empty slot can be chosen for your very first Prg.

Adding a Shortcut to a Prg

The easist way is to just drag in a file from your preferred location. Alternatively, consider leaving Shortcut Nav. (Dlg) unchecked if an actual shortcut, or lnk file is to be used instead of a program file. Then type in a name for the Prg (optional), then click the Make Shortcut button underneath to navigate to the location of the file. For example, navigate to Windows Accessories from Programs in the dialog box, and select the Notepad shortcut, alternatively, navigate to C:\Windows to locate Notepad.exe.

Configure Prg Settings

If desired, change the monitor & resolution, and opt for TestMode along with other selections of your choice. Here, the Prg will launch at below normal priority. In the following graphic, wini.ini is opened as an ANSII file with the text in Command Line Extras. Later on, we''ll see what happens when opening it with a /w switch instead!

Test Run the Prg

Click Test Run Prg to execute it, noting it can be terminated by either clicking the Cancel Prg button, or using its own exit buttons or menu commands. If the operation was successful, select a radio option other than TestMode to set Prg to the desired resolution mode for future runs.

Add in More Prgs

Repeat the above process by adding in other Prgs to the list, in no particular order. For programs known to run without issue, omit the TestMode & Test Run Prg steps and set the Res Option at the recommended Temporary.

Batch Preset Preliminaries

It's possible to associate a Batch Preset with just one Prg, but many situations require programs to be launched in some ordered sequence. Later we'll demonstrate two Prgs running in a batch. Click Back To Prglnch and select a Batch Preset, and optionally assign it a Preset Name.


Configure a Batch Preset

Choose a Preset and some Prgs from the Batch Prg List.The selected Prg names will be prefixed with a number surrounded by asterisks. The number denotes the monitor they will be launched on, and the up-down arrows to the right determine the order of launching, the top in the list being the first. Click Preset Properties to view the PrgLnch Properties.


Starting a Batch Preset

Click Run Batch to begin, Cancel Batch to end. Individual Prgs can stopped/started by double-clicking their corresponding entries in the Prg Status List.
Note: It's also possible to select Prgs from the status list before selecting a Preset, but in applying the Preset, any previous allocations to it are automatically erased.

Creating a LnchPad Slot

Type a name in the LnchPad Slot and click the save button to create the file. For Bethesda Game tragics, there's Lnch Pad presets available for a selection of games on the click of LnchPad Config.

Updating a Prg

A rudimentary example of an updater is setup with Bigger Directories. The URL https://www.StearnVault.com/OldSite/files/BiggerDirectories.exe can be pasted into the Prg's Url Name box, and is saved when the Save URL button is activated. Bigger Directories can then be updated whenever Update Prg is next clicked.

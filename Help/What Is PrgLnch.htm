What Is PrgLnch?

P r g Ln c h is an options based utility that launches programs in various resolutions and in various monitors. A Prg (perhaps the acronym Prefabulated reticulating gestalt qualifies?) might be anything with an *.exe; *.bat; *.com; *.cmd; *.pif; *.msc; *.lnk extension that can be run singly or as part of a batch sequence.

The minimal dialog and overhead of P r g Ln c h is geared for performance and stability, engendering maximum focus on the Prgs as they run in various settings.

Features

  • Monitors status of executing programs.
  • Change monitor resolution - or run programs in a different resolution.
  • Run screensavers or play movies (not in browser) in multimonitor environments through the batch routine.
  • Provides extended information regarding programs in a batch preset.
  • Support for programs updatable at some URL, particularly executables.
  • Toggles the visibility of borders of a an executing program.
  • LAA patch for older programs which may require extra memory.

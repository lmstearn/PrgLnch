Active Window Properties

Displays controls for windows of an active prg selected in the Prg Status List.

Prg Priority:
As in Prg Configuration, the priority setting applies to all windows of the actve prg. Default value is N (Normal). Check the box for H (High) priority or clear it for BN (Below Normal).
Window Minimised, Normal, or Maximised:
As in Prg Configuration, the default value is Norm (Normal) for its saved window positions and dimensions, else it shows as maximised or minimised.
Borderless:
As in Prg Configuration, the regular and optionally extended borderless styles are available for toggle.

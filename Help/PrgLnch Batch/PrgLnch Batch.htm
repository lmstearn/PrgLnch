PrgLnch Batch

Preset Properties: Gets PrgLnch Properties of a selected Batch Preset. Closed when the Preset is de-activated.
Preset Name: Type in a Batch Preset Name corresponding to this Preset. The Del key deletes all contents of the edit box.
List of Batch Presets: Select a preset will commit the preset to the .ini file slot on disk if it exists. If the .ini file slot is empty the current Preset list is written to it. Deselect a preset, (by clicking it again), to remove the corresponding Preset info from the .ini file. The status of a Prg allocated to the Preset which is selected from the adjacent Batch Prgs List, is also displayed in the Prg Status List.
This Preset at Load: When checked, the current Preset is shown and selected when PrgLnch starts.
Batch Prgs List: Select a Prg to add it to a selected Preset and create a corresponding entry in the Prg Status List. The number within the askerisks indicates the monitor configured for the Prg. Deselect it to remove the Prg from the Preset. The Up/Down spin control to the right of the list moves the Prg up and down to change its order in the batch. The selected item highest in the list will be the first Prg to run in the batch, the lowest will be the last Prg run.
Prg Status List: Double click a Prg to start or terminate it. If a Prg is an active process before Prglnch starts, there will be a prompt, and if required, the active Prg will be updated on the Prg Status list. If there are multiple instances of Prg, PrgLnch will choose the one with the lowest PID. If a second instance of Prg is created from a source other than PrgLnch while Prg is active, PrgLnch will ignore it.
Prg Lnch Interval: Default setting is a medium wait between launch of Prgs in the batch. Check the box for a longer wait, clear it for a short wait.
Power Plans: A Power Plan can be applied for an actve Batch. The Power Plan used by the OS before PrgLnch loads is the first one in the list, the Default, which corresponds to one of the following plans in the list.
A Batch Power Plan will be replaced by the Power Plan of the most recent concurrent Batch activated. When PrgLnch starts, a Power Plan for a Batch detected as already active actually remains the Default plan, contrary to the list selection, and will change only when reselected from the list. All Power Plans revert to Default when PrgLnch is quit, even when Batches are still active.
Run Batch: Runs the Prg(s) in the specified Preset. After the Preset is successfully launched, it can be cancelled by clicking this button again. More than one Preset can be run concurrently.
Prg Config: Launches the Prg Configuration option dialog.
LnchPad Slots: Up to 12 slots are provided as LnchPads for programs related to a certain project, game, or schedule of choice. A slot configured for such a purpose is manifest within its ini settings file in the directory PrgLnch is run from. The last used LnchPad is loaded each time PrgLnch starts.
LnchPad Config: A WIP dialog that can prefill slots for a selection of games based on modding tools for various games. On return to PrgLnch, any newly created LnchPad Slots are to be found on the above list.
Quit_PrgLnch: Quits the program. As the close buttons are disabled on both of the PrgLnch Batch and PrgLnch Config dialogs, it's the only way to exit PrgLnch without error.
